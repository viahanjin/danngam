### 사용자 인터랙션

- **← 뒤로**: MainScreen 또는 이전 화면으로 복귀
- **공유**: 카카오톡, 문자 등으로 장비 정보 공유
- **❤️ 찜**: 찜 추가/제거 (토글)
- **이미지 슬라이더**: 좌우 화살표 또는 스와이프로 이미지 변경
- **[더보기]**: 설명 전문 보기 (모달 오픈)
- **[전화]**: 공급자에게 전화 (시스템 전화앱 열기)
- **[채팅]**: 공급자와 채팅 (아직 미구현, 향후 기능)
- **날짜 클릭**: 달력 오픈하여 다른 날짜 가용성 확인
- **[예약하기]**: BookingScreen으로 이동
- **[모든 리뷰 보기]**: 전체 리뷰 목록 화면으로 이동

**단계 1: 날짜 선택**

- 실시간 가격 계산 표시

- [다음: 결제] 클릭 → PaymentScreen으로 이동
