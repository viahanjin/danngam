- **[작성하기]**: 리뷰 제출

## 개발 시 주의사항

1. **아바타 이미지**: 원형으로 자동 자르기 (borderRadius 50%)
2. **글자 수 제한**: TextArea에 maxLength 적용
3. **폼 검증**: 필수 필드 확인 후 제출 버튼 활성화
4. **이미지 업로드**: 압축 후 업로드 (performance)
5. **상태 복원**: 뒤로 가기 시 이전 화면 상태 유지
